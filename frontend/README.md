# Frontend - Vite React Todo App

## Ejecutar en local
1. Copiar `.env.example` a `.env` y ajustar `VITE_API_URL` al backend.
2. Instalar dependencias:
   ```
   npm install
   ```
3. Ejecutar en modo desarrollo:
   ```
   npm run dev
   ```
4. Build para producci√≥n:
   ```
   npm run build
   ```
